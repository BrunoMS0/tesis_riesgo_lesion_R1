Herramientas, métodos y procedimientos de los resultados:

* **R1:** Un modelo de datos formalizado y un conjunto definido de características de ingeniería (features)

  * Herramientas: Jupyter Notebook, Python
  * Método y Procedimiento: Ingeniería de Características (Feature Engineering)

* **R2:** Pipeline de software para la Extracción, Transformación y Carga (ETL) de datos de múltiples fuentes

  * Herramientas: Python, TensorFlow, TensorFlow Data
  * Método y Procedimiento: Proceso ETL (Extracción, Transformación y Carga)

* **R3:** Un dataset final, curado, normalizado y listo para el modelado

  * Herramientas: Google Drive, Python, TensorFlow Record, Google Sheets
  * Método y Procedimiento: Análisis Exploratorio de Datos (EDA)

* **R4:** Un modelo de Deep Learning implementado para el análisis de la fatiga

  * Herramientas: Python, TensorFlow/Keras
  * Método y Procedimiento: Análisis de Series de Tiempo, Redes Neuronales Recurrentes (LSTM/GRU)

* **R5:** Un modelo de Machine Learning implementado para la predicción del riesgo de lesión

  * Herramientas: Python, TensorFlow/Keras
  * Método y Procedimiento: Modelos de Ensamblado (Random Forest/XGBoost)

* **R6:** Un sistema predictivo integrado que orquesta la ejecución de los dos modelos en secuencia

  * Herramientas: Python, Git/GitHub
  * Método y Procedimiento: Integración de Modelos (Model Pipelining)

* **R7:** El modelo de análisis de la fatiga entrenado y validado.

  * Herramientas: Matplotlib, Tensor Board

  * Método y Procedimiento: Validación Cruzada, Métricas de Regresión (RMSE, MAE)

* **R8:** El modelo de predicción de riesgo de lesión entrenado y validado.

  * Herramientas: Tensor Board, Seaborn

  * Método y Procedimiento: Validación Cruzada, Métricas de Clasificación (AUC-ROC)

* **R9:** Un informe técnico de rendimiento y métricas de validación del sistema integrado.

  * Herramientas: LaTeX
  * Método y Procedimiento: Reporte Técnico y Análisis de Resultados

* **R10:** Un análisis de la importancia de características (feature importance) para la predicción de lesiones.

  * Herramientas: SHAP (SHapley Additive exPlanations)

  * Método y Procedimiento: Análisis de Interpretabilidad de Modelos (XAI)

* **R11:** Una validación comparativa del rendimiento del modelo integrado frente a un modelo base.

  * Herramientas: Python, Google Sheets, Matplotlib, Google Drive

  * Método y Procedimiento: Benchmarking de Modelos, Pruebas de Hipótesis
